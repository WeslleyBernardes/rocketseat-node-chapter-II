# Start

```sh 
npm init -y 

# Inicializa typescript
npx tsc --init
# compila
npx tsc 
# executa server.js
node dist/server.js
# inicializa aplicação
npm run dev
```

# Packages

```sh 
npm i express

npm i @types/express

npm i typescript --save-dev

npm i ts-node-dev -D

npm i uuid

npm i @types/uuid -D
```
